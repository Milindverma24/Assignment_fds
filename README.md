## 📌 Project Overview

This project focuses on applying fundamental data science techniques on a network traffic dataset. The workflow includes data preprocessing, statistical analysis, exploratory data analysis (EDA), and implementation of multiple machine learning models.

---

## 🎯 Objectives

- Understand and analyze the dataset  
- Perform data preprocessing and cleaning  
- Apply statistical techniques  
- Conduct exploratory data analysis  
- Implement and evaluate machine learning models  

---

## ⚙️ Technologies Used

- Python 🐍  
- Pandas & NumPy  
- Matplotlib & Seaborn  
- Scikit-learn  

---

## 🔍 Tasks Performed

### 🔹 Task 1: Data Understanding
- Dataset description  
- Data structure analysis  
- Summary statistics  
- Correlation analysis  
- PCA  

### 🔹 Task 2: Statistical Analysis
- Distribution plots  
- Normality testing  
- Hypothesis testing (T-test)  
- Relationship analysis  

### 🔹 Task 3: Data Preprocessing
- Missing value handling  
- Duplicate removal  
- Encoding categorical data  
- Feature scaling  
- Dimensionality reduction  

### 🔹 Task 4: Exploratory Data Analysis
- Histogram  
- Boxplot  
- Scatter plot  
- Heatmap  

### 🔹 Task 5: Machine Learning Models
- Linear Regression  
- Logistic Regression  
- Decision Tree  
- Random Forest  
- K-Means Clustering  

---

## 📊 Model Evaluation

| Model | Evaluation Metrics |
|------|------------------|
| Linear Regression | MSE, R² Score |
| Logistic Regression | Accuracy, Precision, Recall, F1-score |
| Decision Tree | Accuracy |
| Random Forest | Accuracy |
| K-Means | Silhouette Score |

---

## 🧠 Key Insights

- Data preprocessing significantly improved model performance  
- Random Forest achieved the best results among classification models  
- Logistic Regression provided stable and interpretable results  
- K-Means helped in identifying patterns in the dataset  

---

## 🏁 Conclusion

This project demonstrates the complete data science pipeline from raw data processing to model evaluation. Among all models, **Random Forest** performed the best due to its robustness and high accuracy.
